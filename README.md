# Supreme Trade Journal — Mark II

A clean rebuild with Dashboard, New Trade, Trade Review, Analytics, local browser storage, options P&L calculation, search, deletion, and JSON export.

## Open it
1. Unzip the folder.
2. Open the full folder in VS Code.
3. Open `index.html` in Safari or Chrome.
4. Log a test trade.

## Important
Your trades are stored in that browser's localStorage. Use **Export Data** regularly for backups.
